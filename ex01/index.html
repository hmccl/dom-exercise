Exercício de DOM

Questão 1

Estude o que é o DOM e a que ele se aplica.

O Document Object Model (DOM) é uma representação estruturada de documentos HTML e XML que permite aos programadores interagir dinamicamente com o conteúdo, estrutura e estilo de páginas web através de linguagens como JavaScript.

Quando um navegador carrega uma página web, ele converte o código HTML em uma árvore hierárquica de objetos, onde cada elemento HTML (como div, p, img) torna-se um nó no DOM. Esta estrutura em árvore facilita a navegação e manipulação dos elementos.

O DOM funciona como uma interface de programação que oferece métodos e propriedades para acessar e modificar elementos. É possível usar document.getElementById() para selecionar um elemento específico, alterar seu conteúdo com innerHTML, modificar estilos através de style, adicionar ou remover classes, e criar novos elementos dinamicamente.

Esta funcionalidade permite que desenvolvedores respondam a eventos do usuário (cliques, digitação, movimento do mouse), atualizem conteúdo sem recarregar a página, validem formulários em tempo real, e criem animações.
