Exercício de DOM

Questão 5

Qual a diferença entre as propriedades textContent, innerText e innerHTML dos elementos HTML? Cite exemplos.

textContent

Retorna ou define todo o texto dentro de um elemento incluinod o texto oculto por estilos CSS. Não interpreta HTML, apenas manipula texto puro.

<div id="exemplo1">
    <p>Texto visível</p>
    <p style="display:none;">Texto oculto</p>
</div>

<script>
    var div = document.getElementById("exemplo1");
    console.log(div.textContent); // Exibe "Texto visívelTexto oculto"
</script>

innerText

Retorna ou define o texto vísivel no elemento, respeitando o CSS (ou seja, não inclui elementos ocultos). Também não interpreta HTML.

<div id="exemplo2">
    <p>Texto visível</p>
    <p style="display:none;">Texto oculto</p>
</div>

<script>
    var div = document.getElementById("exemplo2");
    console.log(div.innerText); // Exibe "Texto visível" (não inclui o oculto)
</script>

innerHTML

Retorna ou define o código HTML interno do elemento. Permite manipular estruturas HTML dentro de um elemento.

<div id="exemplo3">
    <p>Texto original</p>
</div>

<script>
    var div = document.getElementById("exemplo3");
    div.innerHTML = "<strong>Texto em negrito</strong>"; // Insere HTML
</script>
